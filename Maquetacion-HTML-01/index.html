Introducción

Para este trabajo consulté la hoja de etiquetas disponible de la versión 5 en la página de HTML. documentacion html. Y me apoyé del siguiente sitio: Clic para ver sitio

HTML5 es un lenguaje de marcado donde se hace uso de etiquetas y atributos para dar estructura al texto. En caso de necesitar modificar color o estilo se debe aplicar mediante CSS.

Las etiquetas en html tienen la siguiente estructura: <nombre> contenido </nombre>. En la parte donde va la diagonal invertida indica el fin de la etiqueta. Por ejemplo, el párrafo que estamos escribiendo se encuentra en una etiqueta de párrafo cuyo código se ve de la siguiente manera: <p>texto, texto, texto</p>. Para hacer un salto de linea la etiqueta es <br>.

Para mostrar la estructura de un documento html haremos uso de una lista. Utilizaremos la no ordenada <ul>nombre de la lista</ul> y para cada dato listado debe colocarse dentro de las etiquetas <li>Nombre del elemento</li>:

  • <!DOCTYPE html>:

    Es obligatorio iniciar un documento con esta declaración en HTML5.

  • <html>Documento html</html>:

    Se coloca al inicio y al final para indicar que se trata de un documento html.

  • <head>metadatos</head>:

    Proporciona información general de la página y sirve para colocar etiquetas de metadatos; como el título de la página web. En este ejercicio nuestro título es: "Titulo mostrado en la pestaña".

  • <body>Párrafo</body>:

    Es el cuerpo del documento html y dentro de esta se pueden insertar titulos o subtitulos, asi como párrafos o listas.
