# Practicas-Universidad
Repositorio destinado a almacenar las prácticas, tareas y ejercicios de la carrera de desarrollo de sistemas web.

1. Maquetacion HTML 01: Es un sitio web compuesto solo de uso de etiquetas HTML donde se muestra el contenido.
   [🔗 Ver sitio web en vivo](https://mkowoj.github.io/Practicas-Universidad/Maquetacion-HTML-01/)

2. Canvas HTML: Uso de la etiqueta para dibujar en un lienzo la animación de un círculo.
   [🔗 Ver sitio web en vivo](https://mkowoj.github.io/Practicas-Universidad/Canvas-HTML/)

3. Logica-JS-Estacionamiento: Ejercicio práctico desarrollado en 2024 para comprender la manipulación de estructuras de datos (Arreglos) en JavaScript.
* **Conceptos aplicados:**
  * Métodos de arreglos (`.shift()` para eliminar al inicio, `.pop()` para eliminar al final, modificación por índice).
  * Ciclos e iteraciones (`for` y `while`) para recorrer espacios de almacenamiento.
  * Funciones de búsqueda y condicionales para interactuar con entradas del usuario (`prompt` y `alert`).
* [🔗 Probar simulador en vivo](https://mkowoj.github.io/Practicas-Universidad/Logica-JS-Estacionamiento/)
